Client class for interacting with the Gomoku game server.

Method __init__ Initializes the client with default values, such as username and server URL.
Method check_game_history Interactively shows which moves were made in selected game. Player may choose specific moves.
Method create_game Creates a new game session and retrieves its game ID from the server.
Method get_ids_all_games Displays ids of all games user played
Method get_server Prompts the user to select a server or enter a custom server URL.
Method join_game Joins user to an existing game with specified ID
Method login Logs in an existing user by verifying credentials with the server.
Method make_move Interactively asks player for row and column of place where theirs' piece should be placed. Validated the move.
Method register Registers a new user by sending their credentials to the server.
Method slogan_gomoku Displays the ASCII art slogan of the game.
Method view_board Print an ASCII Picture of current game state including players pieces
