Gomoku CLI game
==========

How to play
-----------

To play on the official server only **gomoku.py** module required.

Launch on Windows:
```
py gomoku.py
```

Mac/Unix:
```
python3 gomoku.py
```

Custom server
-------------

For server **gomoku-server.py** module is needed to be launched.

Launch on Windows:
```
py gomoku-server.py
```

Mac/Unix:
```
python3 gomoku-server.py
```
